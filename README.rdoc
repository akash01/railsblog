== README

This is simple blog application built with ruby on rails 4, bootstrap 3 and pagination support


* Make sure u have ruby version 2 or above and rails version 4 or above

* Download the app

* Change the database configuration in database.yml

* Go to the root of the app folder and run rake db:migrate

* Now run rails s, the app should be up and running at localhost:3000

* Demo at http://rails4-blog.herokuapp.com/
